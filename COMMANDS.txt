üêõ TROUBLESHOOTING
------------------
Issue: Web UI not loading
- Check terminal shows "Web UI started at http://localhost:8000"
- Make sure terminal is not closed
- Try refreshing browser
- Check port 8000 is not in use by another app

Issue: Build errors
- Make sure MinGW GCC is installed and in PATH
- Run: gcc --version (should show version)
- Clean and rebuild: .\build.bat

Issue: Data not showing
- Check data files exist in /data folder
- Verify rides.txt has valid data
- Look for errors in terminal output
