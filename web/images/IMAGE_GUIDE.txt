IMAGE TIPS:
-----------
- Use bright, colorful images with good lighting
- JPG format recommended for photos
- PNG format for images with transparency
- Keep file sizes under 500KB for fast loading
- Use landscape orientation for ride images
- Avoid images with text overlays


WHERE TO FIND FREE IMAGES:
---------------------------
- Unsplash.com (search: amusement park, roller coaster, ferris wheel)
- Pexels.com (search: carnival, theme park, rides)
- Pixabay.com (search: fairground, fun fair)


EXAMPLE SEARCH TERMS:

- "amusement park entrance"
- "roller coaster ride"
- "ferris wheel sunset"
- "carousel horses"
- "drop tower ride"
- "pirate ship ride"
- "theme park aerial view"
- "carnival rides night"


The UI will automatically:
- Use specific images based on ride names
- Fall back to thrill-level images if no name match
- Use default-ride.jpg if no match found
